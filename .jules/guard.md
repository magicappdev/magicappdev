## Security Considerations

### Authentication

- Discord OAuth integration
- Session token storage (use expo-secure-store, not AsyncStorage)
- API token handling

### Data Protection

- User credentials and PII
- API keys and secrets
- OAuth tokens
- Environment variable management

### Third-Party Integrations

- Discord API
- Cloudflare (Workers, KV, R2, D1)
- Expo (builds, updates)
